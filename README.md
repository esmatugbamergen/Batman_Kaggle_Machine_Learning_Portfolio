🚀 Kurulum ve Kullanım / Installation & Usage
TR: Projeleri yerelinizde çalıştırmak için aşağıdaki adımları izleyin:
EN: Follow these steps to run the projects locally:

# Depoyu klonlayın / Clone the repo
git clone https://github.com/esmatugbamergen/Batman_Kaggle_Machine_Learning_Portfolio.git

# Klasöre girin / Enter the directory
cd Batman_Kaggle_Machine_Learning_Portfolio

# Gerekli kütüphaneleri kurun / Install requirements
pip install -r requirements.txt

# Uygulamayı başlatın / Run the app
streamlit run app.py
